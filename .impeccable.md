# Design Context

## Target Audience
Tech professionals and students looking for jobs/internships at VC-backed startups. Skews young — early career engineers, new grads, interns.

## Use Cases
- Search and filter 10K+ jobs by role, location, seniority, salary, VC backer
- Quickly scan dense listings to find relevant opportunities
- Share filtered views via URL

## Brand Personality
Dense and powerful yet professional/refined. Think Bloomberg Terminal meets Linear — information-rich without feeling cluttered. Every pixel earns its place.

## Distinctive Element
Personalization — the filtering and search should feel like the site adapts to YOU. Fast, responsive, precise filtering that makes 10K jobs feel manageable.

## Constraints
- Next.js App Router + Tailwind CSS
- Dark theme
- Table-row layout (dense, sortable)
- Top filter bar with dropdowns
- Must work on mobile
